# Phase 10 — Capstone & Portfolio

**Duration:** Weeks 23–24
**Goal:** Ship one thing so good it speaks for you at interviews.

## Pick one archetype
- **Vertical RAG agent** — Q&A over codebase/legal docs/your own papers
- **Multi-modal assistant** — voice + vision + text
- **Agentic tool** — does real work (refactors, books travel, researches companies)
- **Research-flavored** — reproduce + improve a recent paper
- **Domain-specialized fine-tune** — 7B model that beats base models on your benchmark

## Non-negotiables
1. Public GitHub repo with a great README (problem, demo gif, how to run, architecture, eval results)
2. Deployed demo (HuggingFace Spaces / Fly.io / Railway — all free)
3. Evaluation with real metrics
4. Blog post explaining design decisions

## Week 23 — Build
- [ ] Scope tightly — cut scope in half, then in half again
- [ ] Ship end-to-end ugly version by day 3
- [ ] Iterate on the parts that move the metric
- [ ] Write evaluation harness BEFORE optimizing

## Week 24 — Polish + Ship
- [ ] Recruiter-audience README rewrite
- [ ] Architecture diagram (Excalidraw, Mermaid)
- [ ] 90-second Loom demo
- [ ] Blog post on dev.to / your site
- [ ] Post on LinkedIn, X, HN Show HN
- [ ] Update resume: AI Engineer, not Python developer
- [ ] Apply to 10 roles
