## 🧠 How to use this vault

1. **Work phase by phase.** Open a phase MOC, it lists everything to read + build that week.
2. **Take notes.** When you learn a new concept, either (a) update an existing note in your own words or (b) create a new atomic note and link it.
3. **Link liberally.** The magic of Obsidian is the graph — more links = more pattern recognition later.
4. **Weekly reflection.** Each Sunday, fill in the week's journal note (`journal/Week XX.md`) using [[templates/Weekly Reflection Template|the template]].
5. **1 paper/week minimum** once you hit Phase 5. Use [[templates/Paper Note Template|the paper template]].
