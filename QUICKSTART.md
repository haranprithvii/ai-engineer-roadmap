# Quickstart — 5 minutes to your first week

## 1. Clone this repo (or fork it to your GitHub first)
```bash
git clone https://github.com/YOUR-USERNAME/ai-engineer-roadmap.git
cd ai-engineer-roadmap
```

## 2. Open the tracker
```bash
# macOS
open tracker.html

# Linux
xdg-open tracker.html

# Windows
start tracker.html
```
Your progress is saved in your browser (localStorage). Export to JSON anytime via the "Export progress" button.

## 3. Open the Obsidian vault
1. Download [Obsidian](https://obsidian.md/) (free)
2. Click **Open folder as vault**
3. Pick the `obsidian-vault/` folder inside this repo
4. Start at `00-START-HERE.md`
5. Press `Cmd/Ctrl + G` to open the **Graph view** — this is the magic

## 4. Host your own public version on GitHub Pages (free, 2 min)
1. Fork this repo to your GitHub
2. Go to **Settings → Pages**
3. Under "Build and deployment":
   - Source: **Deploy from a branch**
   - Branch: `main`, Folder: `/website`
4. Click **Save**
5. After ~1 minute your site is live at `https://<your-username>.github.io/<repo-name>/`

Now share the link with your friends.

## 5. Start Phase 0 right now
Open `tracker.html`, expand Phase 00, check off tasks as you go.

Then: open [[obsidian-vault/phases/Phase 00 - Toolchain]] and read the resources.

Welcome. Let's build.
